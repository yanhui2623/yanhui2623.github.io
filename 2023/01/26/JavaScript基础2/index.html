    流程控制（顺序、分支、循环）

    • 分支结构：根据不同的条件，执行不同的路径代码，从而得到不同的结果（if、switch）
    • 三元表达式 var result = 条件表达式 ? 表达式1 : 表达式2;
      如果条件表达式结果为真，则返回表达式 1 的值，为假返回表达式 2 的值
    • switch 语句
      针对变量设置一系列的特定值。1.利用表达式的值和 case 后面的选项值相匹配，如果匹配上，就执行该 case 里面的语句，如果没有匹配上，那么就执行 default 里面的语句 2. 表达式的值和 case 里面的值相匹配的时候时全等（值和数据类型一致）3. 如果 case 里面没有 break，则会执行下一个 case 不会退出 switch
    • switch 语句和 if else if 语句的区别：1. switch…case 通常处理 case 为确定值的情况，if else if 常用于范围 2. switch 语句进行条件判断后直接执行到程序的条件语句，效率更高，而 if else if 语句有几种条件，就判断多少次 3. 当分支较少时，if else if 语句执行效率比 switch 语句高，当分支较多时，switch 语句执行效率比较高，而且结构更清晰
    • 循环 目的：重复执行某些代码

    for 循环执行顺序：1. 初始化变量到条件表达式 2. 条件表达式到循环体 3. 循环体到操作表达式 4. 操作表达式到条件表达式 5. 条件表达式再到循环体

    • 断点调试：1. 浏览器中按 F12 2. sources 3. 找到需要调试的文件 4. 在程序的某一行设置断点 5. F11 单步执行，通过 watch 监视变量的变化

    -双层 for 循环：外层循环一次，里层循环执行全部

    • while 循环：当条件表达式结果为 true，则执行循环体，否则退出循环
    • do while 循环：先执行一次循环体，再判断条件，至少会执行一次循环体
    • continue 关键字：只要遇见 continue 就退出本次循环，直接跳到 i++（退出本次循环）
    • break 关键字：立即跳出整个循环

    数组

    • 存储在单个变量下的一组数据的集合（任意类型）
    • 创建数组
      利用 new 创建数组 var arr = new Array(); 创建一个空数组
      利用数组字面量创建数组 var arr = [];
    • 获取数组元素
      索引：用来访问数组元素的序号（数组下标从 0 开始）
      我们可以通过 数组名[索引号] 的形式来获取数组中的元素
      如果没有该数组元素，输出的结果是 undefined
    • 遍历数组：就是把数组元素从头到尾访问一次
    • 数组长度：数组名.length 动态监测数组元素的个数
    • 数组新增元素：1. 修改 length 长度 var arr = ['red','green','blue']; arr.length = 5;，新增值都是 undefined 2. 修改索引号追加数组元素 var arr = ['red','green','blue']; arr[3] = black;
    • 冒泡排序：是一种算法，把一系列的数据按照一定的顺序进行排列显示（从小到大或从大到小）一次比较两个元素，如果它们顺序错误就把它们交换过来

    函数

    • 封装了一段可以被重复执行调用的代码块，函数间可以相互调用，每个函数都是独立的代码块，用于完成特殊任务，因此会相互调用
    • 函数声明并调用：只有调用函数时才会执行函数体的代码
    • 函数的参数
    • 函数实参形参匹配问题
    • 函数返回值 return：只要遇到 return，就把后面的结果返回给函数调用，相当于 函数名() = return 后面的结果
    • return 终止函数
      return 后面的代码不会被执行
      return 只能返回一个值（最后一个），可以用数组代替
      如果有 return 返回 return 后面的值，没有 return 返回 undefined
    • arguments：当不确定有多少个参数传递的时候，可以用 arguments 来获取
      arguments 实际上是当前函数的一个内置对象，所有函数都内置了一个 arguments 对象，arguments 对象中存储了传递的所有实参
      arguments 展示形式是一个伪数组，可以遍历，具有 length 属性，按索引方式存储，但是不具有数组的方法
    • 函数的两种声明方式

    作用域

    • 指变量在某个范围内起作用和效果，提高程序的可靠性减少命名冲突
    • 全局作用域：整个 script 标签或一个单独的 js 文件
    • 局部作用域：在函数内部就是局部作用域，只在函数内部起效果和作用
    • 块级作用域：{ }，es6 新增，var 定义的变量没有块级作用域的概念，let 和 const 定义的变量在块级作用域以外不可以访问
    • 根据作用域的不同变量可分为全局变量和局部变量

    全局变量：在全局作用域下的变量，在全局下都可以使用，如果在函数内部，没有声明直接赋值的变量也属于全局变量，全局变量只有在浏览器关闭的时候才会销毁，比较占内存
    局部变量：在局部作用域下的变量（函数内部），函数的形参可以看作是局部变量，局部变量只能在函数内部使用，当其所在的代码块运行时，会被初始化，当代码运行结束后，就会被销毁，因此更节省内存空间

    • 作用域链：在内部函数可以访问外部函数的变量，用链式查找决定哪些数据能被内部函数访问，称为作用域链
    • 预解析：javascript 解析器在运行 javascript 代码时候分两步，分为预解析（js 引擎会把 js 里面的所有 var、function 提升到当前作用域的最前面）和代码执行（按照代码书写顺序从上往下执行）
      预解析分为变量预解析（变量提升）和函数预解析（函数提升）
