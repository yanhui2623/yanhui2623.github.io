    浏览器引擎

    • 浏览器分成两部分：渲染引擎和 JS 引擎
    • 渲染引擎：用来解析 HTML 与 CSS，俗称内核，比如 chrome 浏览器的 blink，老版本的 webkit
    • JS 引擎：也称 JS 解释器，用来读取网页中 JavaScript 代码，对其处理后运行，比如 chrome 浏览器的 v8，浏览器本身并不会执行 JS 代码，而是通过内置 JS 引擎来执行 JS 代码，JS 引擎执行代码时逐行解释每一句源码（转换为机器语言），然后由计算机去执行，所以 JavaScript 语言归为脚本语言，会逐行解释执行
    • JavaScript 分为三部分 ECMAScript（JS 基础语法）、DOM（页面文档对象模型）、BOM（浏览器对象模型）

    JavaScript 引入方式

    • 行内式 <input type="button" value="点我试试" onclick="alert('Hello World')">
    • 内嵌式 <script>alert('Hello World~!');</script>
    • 外部 JS 文件 <script src="my.js"></script>

    注释

    • 单行注释 //
    • 多行注释 /* */

    输入输出语句

    • alert('xx') 警示框
    • console.log('xx') 控制台
    • prompt('xx') 输入框

    变量

    • 用于存放数据的容器，通过变量名获取数据，变量是程序在内存中申请的一块用来存放数据的空间
    • 变量的使用：1. 声明变量 var age; 2. 赋值 age = 18;
    • 变量的初始化：var age = 18; 声明变量并赋值称为变量的初始化，一个变量被重新赋值后，它原有的值就会被覆盖，变量值将以最后一次的赋值为准
    • 声明多个变量
    • 声明变量特殊情况
      只声明不赋值 var sex; console.log(sex); 输出 undefined
      不声明不赋值 console.log(tel); 报错
      不声明只赋值 qq = 110; console.log(qq); 输出 110
    • 变量命名规范：由字母，数字，下划线，美元符号（$）组成，不能以数字开头
    • 变量名称一般用名词，函数名称一般用动词

    数据类型

    • 在计算机中，不同的数据所占用的存储空间是不同的，为了便于把数据分成所需内存大小不同的数据，充分利用存储空间，于是定义了不同的数据类型，在代码运行时，变量的类型是由 js 引擎根据右边变量值来判断的
    • javascript 拥有动态类型，意味着相同的变量可作用不同的类型
      var x = 6; x 是数字型
      var x = 'Bill'; x 是字符串型
    • 数据类型的分类
      简单数据类型：Number、String、Boolean、Undefined、Null
      复杂数据类型：Object
    • 数字型Number：既可以保存整数，也可以保存小数（浮点数） var num = 10;
      八进制：在数字前面加 0 表示八进制（0-7） var num = 010; 输出 8
      十六进制：数字前面加 0x 表示十六进制（0-9 a-f） var num = 0xa; 输出 10
    • 数字型最值
      最大值 alert(Number.MAX_VALUE); 输出 1.7976931348623157e+308
      最小值 alert(Number.MIN_VALUE); 输出 5e-324
    • 数字型特殊值
      alert(Infinity); 输出 Infinity（代表无穷大，大于任何数值）
      alert(-Infinity); 输出 -Infinity（代表无穷小，小于任何数值）
      alert(NaN); 输出 NaN（Not a number 代表任何一个非数值）
    • isNaN() 判断是否为非数字
      console.log(isNaN(12)); 输出 false
      console.log(isNaN('red')); 输出 true
    • 字符串型String：字符串型可以是引号中的任何文本
    • 字符串转义符
      \n 换行符（n 是 newline 的意思）
      \\ 斜杠
      \' 单引号
      \" 双引号
      \t tab缩进
      \b 空格（b 是 blank 的意思）
    • 字符串长度
      var str = 'my name is andy';
      console.log(str.length); 输出 15（符号、空格也算）
    • 字符串拼接
      字符串 + 任何类型 = 新字符串
      console.log('沙漠' + '骆驼'); 输出 沙漠骆驼
    • 拼接加强
      var age = 18;
      console.log('我今年'+ age +'岁了'); 输出 我今年18岁了
    • 布尔型Boolean
      只有两个值 true 和 false，true 的值为 1，false 的值为 0
      console.log(false + 1); 输出 1
    • Undefined 和 Null
      一个声明后没有被赋值的变量会有一个默认值就是 undefined
      console.log(undefined + 'red'); 输出 undefinedred
      console.log(undefined + 1); 输出 NaN
      console.log(null + 'red'); 输出 nullred
      console.log(null + 1); 输出 1
      console.log(true + null); 输出 1
    • 字面量
      数字字面量：8、9、10
      字符串字面量：’前端’、’程序员’
      布尔字面量：true、false

    数据类型转换

    • 使用表单、prompt 获取过来的数据默认是字符串类型的，不能直接进行运算
    • 转换为字符串型
      toString() var num = 1; alert(num.toString());
      String() 强制转换 var num = 1; alert(String(num));
      拼接字符串（隐式转换） var num = 1; alert(num + "");
    • 转换为数字型
    • 转换为布尔型
      Boolean() 函数，代表空、否定的值会被转换成 false，例如：’ ‘、0、NaN、null、undefined，其余的值都转换为 true
      console.log(Boolean('')); 输出 false
      console.log(Boolean(0)); 输出 false

    解释型语言和编译型语言

    计算机不能直接理解任何除机器语言以外的语言，所以必须把程序员所写的程序语言翻译成机器语言才能执行程序。程序语言翻译成机器语言的工具，被称为翻译器，翻译器翻译方式有两种：一个是编译，一个是解释，区别在于翻译的时间点不同，编译器是在代码执行之前进行编译，生成中间代码文件，解释器是在运行时进行及时解释，并立即执行

    标识符、关键字和保留字

    • 标识符：指开发人员为变量、属性、函数、参数取的名字（不能是关键字或保留字）
    • 关键字：指 js 本身已经使用了的字，不能再用它们充当变量名、方法名
    • 保留字：就是预留的“关键字”，可能未来会成为关键字，同样不能使用当做变量名或方法名

    javascript 运算符

    • 运算符（operator）也称为操作符，用于赋值、比较、运算等功能的符号
    • 算术运算符 + - * / % 注意先乘除后加减
    • 浮点数算数运算会有问题
      console.log(0.1 + 0.2); // 0.30000000000000004
      console.log(0.07 * 100); // 7.000000000000001
      var num = 0.1 + 0.2; console.log(num == 0.3); // false
    • 表达式：由数字、运算符、变量等组成的式子，表达式最终都会有一个结果，返回给我们，称为返回值
    • 递增递减运算符（递增递减运算符必须和变量配合使用）

    ++num 先自加，后返回值
    –num 先自加，后返回值
    num++ 先返回原值，后自加
    num– 先返回原值，后自加

    • 对以上代码的解释：可知 e 为 10，e++ 先返回原值，后自加，意思是 e++ 的值 返回 10，再自加 1 这时 e 为 11，但是 e++ 的值就是 10，++e 先自加就是 11 + 1 = 12，后返回值返回的就是 12，最后 10 + 12 为 22
    • 比较运算符 <、 >、 >=、 <=、 ==（等于）、 !=（不等于）、 ===（全等，要求值和数据类型一样）、 !== 两个数据进行比较时所使用的运算符，返回一个布尔值
      console.log(18 == '18'); // true（== 默认转换数据类型）
    • 逻辑运算符：用来进行布尔值运算的运算符，其返回值也是布尔值
      && 逻辑与（and）
      || 逻辑或（or）
      ！ 逻辑非（not）
    • 逻辑中断（短路）：当有多个表达式的值，左边的表达式值可以确定结果时，就不再运算右边的表达式的值
    逻辑运算符 表达式 解释
    逻辑与 表达式1 && 表达式2 如果表达式 1 结果为真，则返回表达式 2
    如果表达式 1 结果为假，则返回表达式 1
    逻辑或 表达式1 || 表达式2 如果表达式 1 结果为真，则返回表达式 1
    如果表达式 1 结果为假，则返回表达式 2
    • 赋值运算符：= 直接赋值、+=、-=、*=、/=、%= 加减乘除取余之后再赋值

    运算符优先级

    优先级 运算符
    1 小括号 ()
    2 一元运算符 ++、–、！
    3 算数运算符 先 */ 后 +-
    4 关系运算符 >、>=、<、<=
    5 相等运算符 ==、!=、===、!==
    6 逻辑运算符 先 && 后 ||
    7 赋值运算符 =
    8 逗号运算符 ,
