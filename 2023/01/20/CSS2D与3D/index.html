    transform(转换)是 CSS3 中具有颠覆性的特征之一，可以实现元素的位移、旋转、缩放等效果
    二维坐标系：x 轴水平向右为正方向，y 轴垂直向下为正方向
    三维坐标系：x 轴水平向右为正方向，y 轴垂直向下为正方向，z 轴垂直屏幕（也就是电脑指向你的方向）为正方向

    2D 转换之位移

    • transform: translate(x,y); 向 x 轴移动 xpx 并向 y 轴移动 ypx
    • transform: translateX(n); 向 x 轴移动 npx
    • transform: translateY(n); 向 y 轴移动 npx
    • transform: translate(50%,50%) translate 中的百分比单位相对于自身元素宽高的 50%
    • 注意：位移不会影响到其他元素的位置，位移对行内标签没有效果

    2D 转换之旋转

    • transform: rotate(度数); 单位是 deg（度数），角度为正顺时针，角度为负为逆时针
    • transform-origin: x y; (默认50% 50%) 设置元素转换中心点 可以用精确单位px或方位名词或百分比

    2D 转换之缩放

    • transform: scale(x,y);
    • 默认 transform: scale(1,1); 宽和高都放大一倍，相当于没有放大
    • transform: scale(2); 只写一个参数，默认第二个参数与第一个参数参数相同
    • transform: scale(0.5,0.5); 缩小
    • transform-origin: x y; (默认50% 50%) 也可以设置旋转中心点缩放，默认以中心点缩放，不影响其他盒子

    2D 转换综合写法

    • transform: translate() rotate() scale();
    • 注意：顺序会影响转换的效果，应将位移放到最前面

    CSS 动画基本使用（animation）

    • 使用动画分为两个步骤：1. 先定义动画 2. 再使用动画
    1. 定义动画：用 keyframes（关键帧） 定义动画：0%（from） 是动画的开始，100%（to） 是动画的完成，百分比就是持续时间的划分
    1. 使用动画两个必须属性 animation-name: 动画名称; animation-duration: 持续时间;
    • 非必须属性，动画的速度曲线 animation-timing-function: ease/linear; 默认为 ease
    • 非必须属性，动画何时开始 animation-delay: 0; 默认为 0
    • 非必须属性，动画播放次数 animation-iteration-count: 1/infinite; 默认为 1
    • 非必须属性，动画在下一周期是否逆向播放 animation-direction: normal/alternate; 默认为 normal
    • 非必须属性，动画是否在运行或暂停 animation-play-state: running/paused; 默认为 running
    • 非必须属性，动画结束后状态 animation-fill-mode: backwards(回到起始)/forwards(保持); 默认为 backwards
    • 动画简写属性 animation: name duration timing-function delay iteration-count direction fill-mode;

    3D 转换之位移

    • transform: translateX(100px); 在 X 轴上移动 100px
    • transform: translateY(100px); 在 Y 轴上移动 100px
    • transform: translateZ(100px); 在 Z 轴上移动 100px
    • transform: translate3d(100px,100px,100px);
    • 透视 perspective: 像素值; 透视写在被观察元素的父元素上面

    3D 转换之旋转（左手定则）

    • transform: rotateX(45deg); 沿着 x 轴正方向旋转 45 度
    • transform: rotateY(45deg); 沿着 Y 轴正方向旋转 45 度
    • transform: rotateZ(45deg); 沿着 Z 轴正方向旋转 45 度
    • transform: rotate3d(x,y,z,deg);
    • transform: rotate3d(1,0,0,45deg); 沿着 x 轴正方向旋转 45 度
    • transform: rotate3d(1,1,0,45deg); 沿着 xy 轴对角线正方向旋转 45 度

    3D 呈现

    • transform-style: flat(默认); 控制子元素是否开启三维立体环境，默认 flat 不开启 3d 立体空间
    • transform-style: preserve-3d; 子元素开启立体空间
    • 注意：代码写给父级，但是影响的是子盒子
