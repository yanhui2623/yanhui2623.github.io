    表格标签(展示数据)

    table 属性名 属性值 描述
    align left（默认）、center、right 相对周围元素对齐方式
    border 1或”” 表格的边框
    cellpadding 像素值 单元边与内容之间的空白，默认为 1px
    cellspacing 像素值 单元格之间的空白，默认为 2px
    width 像素值或百分比 规定表格的宽度
    height 像素值或百分比 规定表格的高度

    表单

    表单：为了收集用户信息，一个完整的表单通常由表单域、表单控件和提示信息组成，表单域是一个包含表单元素的区域，<form></form> 会把它范围内的表单元素信息提交给服务器。

    form 属性 属性值 作用
    action url 地址 用于指定接收并处理表单数据的服务器程序的 url 地址
    method get/post 用于设置表单数据的提交方式，其取值为 get 或 post
    name 名称 用于指定表单的名称，以区分用一个页面中的多个表单域

    <input type="属性值" name="自定义" value="自定义">

    input 属性 属性值 描述
    type 看下个表格 定义各种表单元素控件
    name 自定义 定义 input 元素名称
    value 自定义 定义 input 元素的值
    checked checked 首次加载被选中
    maxlength 正整数 输入字段中字符的最大长度
    type 属性值 描述 注释
    text 文本框
    password 密码框
    radio 定义单选按钮 相同的 name 属性才可以实现多选一
    checkbox 定义复选框 name 属性必须相同
    submit 定义提交按钮 会把表单数据发送到服务器
    reset 定义重置按钮 会清楚表单中的所有数据
    button 定义可点击按钮
    file 定义上传文件按钮
