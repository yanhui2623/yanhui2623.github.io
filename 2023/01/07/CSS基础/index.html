    CSS 选择器

    选择器 示例 在 html 标签添加的属性
    标签选择器 p {}
    类选择器 .类名 {} class=”类名”
    多类名选择器 .类名 {} class=”类名1 类名2”
    id 选择器 #id名 {} id=”id名”
    属性选择器 [属性名] {} 属性名=””
    通配符选择器 * {}
    后代选择器 元素1 元素2 {}
    子元素选择器 元素1>元素2 {}
    并集选择器 元素1,元素2 {}
    链接伪类选择器 a:link 选择所有未被访问的链接
    链接伪类选择器 a:visited 选择所有已被访问的链接
    链接伪类选择器 a:hover 选择鼠标指针位于其上的链接
    链接伪类选择器 a:active 选择鼠标按下未弹起的链接
    伪类选择器 :focus 用于获取焦点的表单元素

    字体属性

    • 字体系列 font-family: Arial,"Microsoft Yahei","微软雅黑";
    • 字体大小 font-size: 20px;（标题标签需要单独修改）
    • 字体粗细 font-weight: normal(默认) 400/ bold 700;
    • 字体样式 font-style: normal(默认) / italic(斜体);
    • 字体复合属性 font: font-style font-weight font-size/line-height font-family;
      注意：字体复合属性不能更换属性，后面两项为必须属性

    文本属性

    • 文本颜色 color: red/#FF0000/rgb(255,0,0);
    • 文本对齐 text-align: center/left(默认)/right;（设置元素内文本内容的水平对齐方式，里面的行内元素或者行内块元素都可以居中对齐）
    • 文本装饰 text-decoration: none(默认)/underline(下划线)/overline(上划线)/line-through(删除线);
    • 文本缩进 text-indent: 2em;
    • 行间距 line-height: 26px;（上间距+文本高度+下间距）

    单行文字垂直居中，就是让文字的行高等于盒子的高度

    CSS 引入方式

    • 内部样式表 <style>样式</style> 只能控制一个页面
    • 行内样式表 style="color: red; font-size: 12px;" 只能控制一个标签
    • 外部样式表 <link rel="stylesheet" href="CSS文件路径"> 可以控制多个页面

    Emmet 语法

    • 输入标签名按 tab 键即可
    • div*3 快速生成 3 个 div
    • 父子关系用> 例如 ul>li
    • 兄弟关系用+ 例如 div+p
    • 生成带有类名或者 id 名字的 直接写 .demo 或者 #two 按 tab 键，默认是 div 标签，如果要用其他的标签 则 p.demo
    • 可以用自增符号 $，例如 p{$}*5
    • 在生成的标签内部写内容可以用 {} 表示，例如 p{内容}

    块元素特点

    • 自己独占一行
    • 高度、宽度、外边距以及内边距都可以控制
    • 宽度默认是容器的（父级宽度）的 100%，也就是不指定宽度和父亲一样宽（通栏的盒子不指定宽度）
    • 是一个容器及盒子，里面可以存放行内或者块级元素

    行内元素特点

    • 相邻行内元素在一行上，一行可以显示多个
    • 宽、高直接设置无效
    • 默认宽度是本身内容的宽度
    • 只能容纳文本或其他行内元素

    行内块元素特点

    • 和相邻行内元素（行内块元素）在一行上，但是它们之间会有空白缝隙
    • 默认宽度就是它本身内容的宽度（行内元素特点）
    • 高度、宽度、外边距以及内边距都可以控制（块级元素特点）

    显示模式的转换

    • 转换为块级元素 display: block;
    • 转换为行内元素 display: inline;
    • 转换为行内块元素 display: inline-block;

    背景属性

    • 背景颜色 background-color: red/#FF0000/rgb(255,0,0); 默认值 transparent 透明的
    • 背景图片 background-image: none/url(地址);
    • 背景平铺 background-repeat: repeat(默认)/no-repeat/repeat-x/repeat-y;
    • 背景位置 background-position: x y; x、y 的值可以是方位名词或者精确单位

    方位名词：top、center、bottom、left、right如果指定的两个值都是方位名词，则两个值的前后顺序无关，如果只指定了一个方位名词，另一个值省略，第二个值默认是 center
    精确单位：像素值，第一个值肯定是 x 坐标，第二个值肯定是 y 坐标，如果只指定一个数值，该数值一定是 x 坐标，另一个默认垂直居中
    混合单位：指定两个值是精确单位和方位名词，第一个是 x 坐标，第二个是 y 坐标

    • 背景固定 background-attachment: scroll(默认)/fixed; 设置背景图像是否随着页面的其余部分滚动
    • 背景复合写法 background: 背景颜色 背景图片地址 背景平铺 背景图像滚动 背景图片位置;
    • 背景色半透明 background: rgba(0,0,0,0.3);

    CSS 三大特性

    • 层叠性 相同的选择器设置相同的样式
    • 继承性 子标签会继承父标签的某些样式，例如 text-、font、line-、及 color 属性
    • 优先级 如下图
    选择器 优先级
    继承或 * 0,0,0,0
    伪元素选择器或元素选择器 0,0,0,1
    属性选择器、类选择器或伪类选择器 0,0,1,0
    ID 选择器 0,1,0,0
    行内样式 style=”” 1,0,0,0
    !important ∞无穷大

    盒子模型

    • 边框粗细 border-width: px值;
    • 边框样式 border-style: solid(实线)/dashed(虚线)/dotted(点线);
    • 边框颜色 border-color: red/#FF0000/rgb(255,0,0);
    • 边框复合写法 border: border-width(边框粗细) border-style(边框样式) border-color(边框颜色);
      上边框 border-top 下边框 border-bottom 左边框 border-left 右边框 border-right
    • 表格细线边框合并 border-collapse: collapse; 表示相邻边框合并在一起
    • 边框会影响盒子的大小：● 测量盒子大小的时候，不量边框 ● 如果测量的时候包含了边框，则需要减去边框
    • 内边距 padding-top、padding-bottom、padding-left、padding-right
    • 外边距 margin-top、margin-bottom、margin-left、margin-right
    • 复合属性同下
    内边距复合属性 解释
    padding: 5px; 代表上下左右都是 5px 内边距
    padding: 5px 10px; 代表上下内边距是 5px，左右内边距是 10px
    padding: 5px 10px 20px; 代表上内边距是 5px，左右内边距 10px，下内边距 20px
    padding: 5px 10px 20px 30px; 代表上内边距是 5px，右内边距 10px，下内边距 20px，左内边距 30px
    • padding 会影响盒子实际大小：让 width/height 减去内边距的大小即可，如果盒子本身没有指定 width/height 属性，则此时 padding 不会撑开盒子大小
    • 块级盒子水平居中对齐：首先盒子必须指定了宽度（width），然后盒子的左右外边距都设置为 auto margin-left: auto; margin-right: auto; 或者 margin: auto; 或者 margin: 0 auto;
    • 行内元素或者行内块元素水平居中给其父元素添加 text-align: center 即可
    • 对于两个嵌套关系（父子）的块元素，父元素有上外边距同时子元素也有上外边距，此时父元素会塌陷较大的外边距值
      解决方案：1. 可以为父元素定义上边框 2. 可以为父元素定义内边距 3. 可以为父元素添加 overflow: hidden; 4. 有浮动、固定、绝对定位的盒子不会有塌陷问题
    • 清楚内外边距
      * { padding: 0; margin: 0; }
    • 行内元素尽量只设置左右内外边距，不要设置上下内外边距，但是转换为块级或行内块元素就可以了
    • 去掉 li 前面的项目符号 list-style: none;

    圆角边框

    • border-radius: px值/百分比;
    • 如果是正方形，想要设置为一个圆，把数值修改为高度或者宽度的一半即可，或直接写 50% border-radius: 50%;
    • 如果是矩形，设置为高度的一半，就可以做圆角矩形
    • 也可以单独设置各个角的大小 border-radius: 左上角 右上角 右下角 左下角;
    • 分开写 border-top-left-radius、border-top-right-radius、border-bottom-left-radius、border-bottom-right-radius

    阴影

    • 盒子阴影 box-shadow: h-shadow v-shadow blur spread color inset;
    阴影属性 备注
    h-shadow 必需，水平阴影的位置，允许负值
    v-shadow 必需，垂直阴影的位置，允许负值
    blur 可选，模糊距离（虚实）
    spread 可选，阴影的尺寸（大小）
    color 可选，阴影的颜色（rgba）
    inset(默认 outset) 可选，将外部阴影（outset）改为内部阴影
    • 文字阴影 text-shadow: h-shadow v-shadow blur color;
