    对象

    • 对象由属性和方法组成，保存一个值可以用变量，保存多个值可以用数组，要保存一个人完整信息就用对象
    • 创建对象的三种方式
    • 调用对象属性的两种方式：
      对象名.属性名
      对象名['属性名']
    • 调用对象方法的方式 对象名.方法名()
    • new 关键字执行过程：1. new 构造函数可以在内存中创建一个空对象 2. this 就指向刚才创建的空对象 3. 执行构造函数里面的代码，给这个空对象添加属性和方法 4. 返回这个对象
    • 遍历对象：for…in 语句用于数组或者对象的属性进行循环操作 for(变量 in 对象名) {}

    内置对象

    • javascript 语言自带的一些对象，供开发者使用
    • Math 对象：不是一个构造函数，不需要 new 来调用，直接使用即可

    圆周率 console.log(Math.PI); // 3.141592653589793
    最大值 console.log(Math.max(1, 99, 3)); // 99
    console.log(Math.max(-1, -10)); // -1
    console.log(Math.max(1, 99, 'red')); // NaN
    console.log(Math.max()); // -Infinity
    绝对值 console.log(Math.abs(-1)); // 1
    console.log(Math.abs('-1')); // 1（隐式转换）
    console.log(Math.abs('red')); // NaN
    三个取整方法
    Math.floor() 向下取整
    console.log(Math.floor(1.1)); // 1
    console.log(Math.floor(1.9)); // 1
    Math.ceil() 向上取整
    console.log(Math.ceil(1.1)); // 2
    console.log(Math.ceil(1.9)); // 2
    Math.round() 四舍五入
    console.log(Math.round(1.1)); // 1
    console.log(Math.round(1.5)); // 2
    console.log(Math.round(1.9)); // 2
    console.log(Math.round(-1.5)); // -1（其他数字正常四舍五入，但是 .5 是往大了取）

    • 随机数 random() console.log(Math.random()); // [0,1)
    • 要得到两个数之间的随机整数并且包含这两个整数
    • 日期对象：是一个构造函数，必须用 new 来创建日期对象
    • 日期格式化
    函数 解释
    getFullYear() 获取当年
    getMonth() 获取当月（0-11）
    getDate() 获取当天日期
    getDay() 获取星期几（周日0到周六6）
    getHours() 获取当前小时
    getMinutes() 获取当前分钟
    getSeconds() 获取当前秒钟
    • 获取日期总毫秒形式（时间戳）：距离 1970 年 1 月 1 日过了多少毫秒数
    • 总毫秒转换为天、时、分、秒（1 秒 = 1000 毫秒）
    • 数组对象：创建数组
    • 检测是否为数组
    • 添加数组元素方法
    • 删除数组元素方法
    • 数组排序
    • 数组索引
    • 数组转换为字符串
    • 数组其他方法
    • 字符串对象
    • 基本包装类型（String、Number、Boolean），把简单数据类型包装成复杂数据类型，这样基本数据类型就有了属性和方法
    • 字符串不可变：指字符串里面的值不可变，看上去改变了内容，但其实地址变了，在内存中新开辟了一个内存空间
    • 根据字符返回位置：字符串的所有方法，都不会修改字符串本身，操作完成会返回一个新字符串
    • 根据位置返回字符
    • 字符串拼接
    • 替换字符
