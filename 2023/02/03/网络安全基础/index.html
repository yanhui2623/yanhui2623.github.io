    • 访问 FTP 服务器：首先打开我的电脑，在地址栏中输入 ftp://10.0.105.223
    • win10 关闭更新服务：win + R 输入 services.msc 找到 Windows Update 双击进入在常规栏中的启动类型改为禁用，恢复栏中的第一次失败改成无操作，再 win + R 输入 gpedit.msc 打开本地组策略编辑器找到计算机配置下的管理模板下的 Windows 组件下的 Windows 更新找到配置自定更新改为已禁用
    • 配置 IP 地址：点击网上邻居并右键再点击属性，再点击本地连接并右键再点击属性（win7/2008：点击网络并右键点击属性，点击本地连接，再点击属性，双击Internet 协议版本 4 (TCP/IPV4)即可配置），可以看到 Internet 协议(TCP/IP)，双击打开选择使用下面的 IP 地址手动修改即可（如何查看：双击本地连接，选择支持选项栏即可查看）

    虚拟机运行架构

    寄居架构（实验环境、测试环境）

    1. 虚拟机作为应用软件安装在操作系统上
    2. 可以在此应用软件上安装多个操作系统
    3. 直接安装在硬件上的系统为宿主

    原生架构（生产环境）

    1. 虚拟机软件直接安装在计算机硬件上
    2. 虚拟机本身就是一个操作系统
    3. 在这个虚拟机中可以同时运行多个操作系统

    虚拟化产品

    Vmware

    1. VMware Workstation（寄居架构）
    2. VMware vSphere（原生架构）

    微软

    • Hyper-V（寄居架构）

    部署虚拟机

    1. 点击创建新的虚拟机（Windows 系统用典型，linux 系统用自定义）
    2. 选择稍后安装操作系统
    3. 选择客户机操作系统（VMware ESX 是原生架构），再选择版本
    4. 起一个虚拟机的名称，并选择虚拟机存放位置（不要放置在 C 盘）
    5. 最大磁盘大小不用改，这里只是设置一个虚拟机内存的最大值，继续选择将虚拟磁盘存储为单个文件
    6. 点击完成就可以了
    7. 点击 CD/DVD（物理驱动器就是光盘，我们选择使用 ISO 映像文件）
    8. 点击开启此虚拟机即可

    安装 Windows xp/2003 系统

    1. 首先选择要在尚未划分的空间中创建磁盘分区，分给 10240MB（10G）即可，选择分区 1 安装 Windows xp （按回车即可，不要选择未划分的空间，如果选择了就是将系统装到了 D 盘，可能会拖慢开机速度）
    2. 选择用 NTFS 文件系统格式化磁盘分区（快）

    安装 Windows 7/2008 系统

    1. 在你想进行何种类型的安装中选择自定义，直接下一步即可，先全部分给 C 盘，在装完系统之后点击此电脑并右键点击管理，再点击磁盘管理，找到 C 盘，右键点击压缩卷，等待压缩完成再点击未分配的区域并右键点击新建简单卷，一直点击下一步即可*
    2. 或者在你想进行何种类型的安装中选择自定义，点击驱动器选项（高级），再点击新建，一般来说给 20480MB 即可

    系统优化

    1. 点击虚拟机，再点击安装 VMware tools 即安装驱动，在弹出的窗口中一直点击下一步即可最后重启系统
    2. 调出桌面图标：右键系统桌面点击属性，再点击桌面、自定义桌面添加即可
    3. 磁盘管理（创建 D 盘分区）：单击我的电脑右键点击管理，再点击磁盘管理，找到未指派的磁盘单击并右键新建磁盘分区选择主磁盘分区，下一步即可，将快速格式化打勾
    4. xp、2003 默认关闭防火墙
    5. 虚拟机克隆：点击快照（管理此虚拟机的快照），点击克隆，选择现有快照，选择创建链接克隆，开机，点击我的电脑并右键点击属性，再点击计算机名会发现计算机名和刚才的一样，更改一下并重启计算机，最后再给这台计算机做快照
    6. win7和2008 关闭防火墙：打开控制面板点击系统和安全，再点击 Windows 防火墙，再点击打开或关闭 Windows 防火墙，最后将家庭或工作网络位置设置和公用网络位置设置都设置成关闭 Windows 防火墙再点击确定即可
    7. win 7/2008 关闭更新：打开控制面板点击系统和安全，再点击 Windows Update，再点击更改设置，找到下拉菜单选择从不检查更新
    8. 做快照：将以上步骤都做完，点击第三个时钟图标（管理此虚拟机的快照），建议关机再做快照

    安装盘制作

    1. 纯净映像下载网站 http://msdn.itellyou.cn ，复制 ED2K 或者 BT 到迅雷中下载
    2. 下载 UltraISO 软件并以管理员身份运行，点击文件再点击打开选择目标 ISO 文件，点击启动再点击写入硬盘映像，只需更改硬盘驱动器为自己想刻入的盘，最后点击写入即可

    解决 win7 或 2008 无法安装 VMware Tools 问题

    1. 点击计算机（xp 或 2003 是点击我的电脑）并右键点击属性，点击远程设置，选择允许运行任意版本远程桌面的计算机连接，点击应用并确定
    2. win + R 打开 cmd 终端，输入 ipconfig ，记录 ipv4的地址
    3. 在主机中 win + R 打开 cmd 终端，输入 mstsc 进入远程桌面（3389 端口），输入虚拟机 ipv4 地址（虚拟机最好设置一个登录密码，否则可能连不上远程桌面）
    4. 补丁下载网站：https://www.catalog.update.microsoft.com/Search.aspx?q=kb4474419 选择所缺的系统补丁下载并复制粘贴到远程桌面，退出远程登陆，重新登录虚拟机
    5. 双击补丁程序并安装，等待安装完成重启虚拟机，之后即可正常安装 VMware Tools
