    网络安全 WEB 服务器

    网络安全 网络安全
    Created At : 2023-02-27 20:10
    Count:1.4k Views 👀 :

    2. 部署 WEB 服务器
    3. 同时部署多个 web 网站
    4. 网站类型
    5. 部署动态网站

    概述

    • WEB 服务器也称为网页服务器或者 HTTP 服务器
    • WEB 服务器使用的协议是 HTTP 或 HTTPS，二者区别在于安全与不安全，HTTPS 协议是经过加密的
    • HTTP 协议端口号 TCP 80，HTTPS 协议端口号 TCP 443
    • WEB 服务器发布软件：微软 IIS（Internet Information Services 互联网信息服务可以发布 web 网站和 FTP 站点），Linux Apache/LAMP/Tomcat/nginx，第三方 phpstudy/XAMPP（供学习测试使用两款软件内置 LAMP，微软和 Linux 在生产环境中使用）

    部署 WEB 服务器

    1. 配置静态 IP 地址：点击网络并右键点击属性，点击本地连接，点击属性，双击 Internet 协议版本 4（TCP/IPv4）即可配置保持计算机之间可以相互通信
    2. 安装 IIS：打开管理工具，服务器管理器，角色，添加角色，然后点击下一步，选择安装“Web 服务器（IIS）”，点击下一步，下一步，安装，等待安装完成点击关闭
    3. 此时打开客户机的浏览器输入服务器的 IP 地址即可访问（默认网站 Default Web Site 文件在 C:\inetpub\wwwroot 目录下，如果默认网站被停止则系统的 80 端口也随之关闭）
    4. 新建自己的网站：打开管理工具，Internet 信息服务（IIS）管理器，计算机名称，点击网站并右键添加网站会弹出对话框：网站名称随便写，绑定类型选择 http，IP 地址选择服务器的 IP 地址，端口为80，物理路径选择装有 html 文件夹，主机名暂时不填写，最后点击确定（在身份验证中可以设置匿名身份验证，默认文档可以更改首页的优先级）
    5. 在客户机浏览器输入 IP 地址即可访问

    同时部署多个 web 网站

    1. 点击网站并右键点击添加网站，按要求输入IP 地址选择服务器的 IP 地址点击确定即可此时服务器会提示不能在一个 IP 上挂两个网站，所以解决办法有两种
    2. 方法一：在服务器上点击网络并右键点击属性，点击本地连接，属性，Internet 协议版本 4（TCP/IPv4），高级，添加 IP 地址和子网掩码，点击添加即可（ipconfig 命令可以查看本地连接下的两个 IP 地址）
    3. 打开 IIS 管理器，计算机名，网站，网站名称，绑定，选中一个选项点击编辑，IP 地址改为新增加的 IP 地址，确定，关闭即可
    4. 打开客户机访问即可（如果访问不了试着更改下默认文档）
    5. 方法二：网站，网站名称，绑定，选中一个选项点击编辑，IP 地址和上一个网站的 IP 相同，端口号改为 8000（随便一个数字，1000之前国家几乎已经预定了）点击确定，客户机访问的时候需要输入 IP 地址:8000 的格式
    6. 方法三：安装 DNS 服务器软件，打开 DNS 管理器，计算机名，点击正向查找区域并右键点击新建区域，下一步，主要区域下一步，yanhui.com下一步，创建新文件下一步，不允许动态更新下一步，完成，再重新创建一个 nba.com 正向解析文件，选中域名文件夹并右键点击新建主机（名称 www，IP 地址为 WEB 服务器的 IP 地址），添加主机，在客户机上指定 DNS 服务器，回到 IIS 管理器，网站，网站名称，绑定，选中一个选项点击编辑，IP 地址和上一个网站的 IP 相同，端口号也相同，主机名一个为 www.yanhui.com 和 www.nba.com 用域名来区分不同的 WEB 网站，在客户机的浏览器上输入 www.yanhui.com 和 www.nba.com 可以打开两个不同的网站

    网站类型

    1. 静态网站：一般扩展名为 .html 或 .htm 无后台数据库
    2. 动态网站：一般扩展名为 .asp 或 .php 有后台数据库，asp 或 php 可以连接前台页面和后台数据库

    部署动态网站

    1. 打开 windows 2003 服务器，我的电脑，CD 驱动器，安装可选的 Windows 组件，双击打开应用程序服务器（不是打勾），双击打开 Internet 信息服务（IIS），将万维网读物打勾，确定，确定，下一步，等待安装完成，此时用客户机浏览器访问服务器的 IP 即可查看默认站点（默认网站 Default Web Site 文件在 C:\inetpub\wwwroot 目录下）
    2. 打开 Internet 信息服务（IIS）管理器，双击本地计算机，点击网站并右键点击新建，选择网站，点击下一步，描述是显示在网站文件夹下的名称，选择网站的 IP 地址，端口默认是 80，此网站的主机头不填，下一步，选择路径下一步，允许权限（如果是静态网站选择第一个即可，这里是动态网站所以选择前三个），点击完成
    3. 点击网站描述并右键点击属性，点击文档，添加该网站的首页文件名并上移到第一位并确定
    4. 点击 Web 服务扩展，找到 Active Server Pages，点击允许
    5. 用客户机浏览器访问即可
    6. 如果要访问后台管理系统：则点击网站描述并右键点击属性，点击主目录，点击配置，点击选项，将启用父目录打勾即可
