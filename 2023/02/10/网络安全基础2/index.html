    测试网络连通性

    ping 目标IP地址 测试网络连通性，有去有回即可以成功通信
    ping -t 目标IP地址 一直 ping
    ping -n 数字 目标IP地址 修改 ping 包的数量
    ping -l 数字 目标IP地址 修改 ping 包的大小
    注意：如果 ping 不通，可能对方不在线，或者对方开着防火墙

    手工解析域名

    nslookup www.jd.com 手工解析域名的 IP 地址

    查看 IP 地址

    ipconfig 查看 IP 地址基本信息
    ipconfig /all 查看 IP 地址详细信息（可以看到 DNS 服务器的信息）

    基本 DOS 命令（Tab 键补全路径）

    注意：/s 为逐级删除 /q 为不要提示 /a 所有文件 .\ 为当前目录
    基本命令
    color f0 改变背景及字体颜色（f 控制背景颜色 0 控制字体颜色 color ? 可以查看颜色序号）
    cls 清屏
    目录相关命令
    dir c:\ 浏览当前 C 盘中的文件
    dir c:\pic 浏览当前 C 盘下 pic 文件夹中的文件（也可以加 | more）
    dir /a 浏览用户文件夹下的所有内容，包括隐藏、受系统保护的内容
    d: 切换盘符
    cd.. 返回上一级
    cd 文件夹名 进入文件夹
    cd \ 直接退到根目录
    md 文件夹 [文件夹 文件夹...] 创建文件夹（多个参数创建了多个同级文件夹）
    rd 文件夹 [文件夹 文件夹...] 删除空文件夹（必须是空文件夹）
    rd 文件夹 /s /q 无提示删除文件夹（有文件也可以删除）
    rd . /s /q 删除所有文件
    文件相关命令
    echo "hhh" 屏幕上输出 hhh
    echo 文件内容 >> [路径\]文件名.扩展名 创建文件（文件名.扩展名相同：>> 追加文件，> 覆盖文件 不同则都是创建文件）
    type 文件名.扩展名 浏览一个文件内容
    type 文件名.扩展名 | more 分页显示文件内容（空格翻页）
    del 文件名.扩展名 删除文件
    del *.txt 删除所有 txt 结尾的文件
    del *.* 删除所有文件
    del *.* /s /q 无提示删除所有文件
    复制和移动
    copy [路径]源文件名 [目标路径]新文件名 复制源文件到目标路径（路径可以是绝对路径或者相对路径）
    copy con 文件名.扩展名 创建文件可以复制屏幕内容到文件名.扩展名（con 屏幕的意思 ctrl + z 回车结束编辑）
    move [路径]源文件名 [目标路径]新文件名 移动源文件到目标路径
    ren [路径]源文件名 新文件名 修改文件名
    定时关机或重启
    shutdown -s -t 秒 定时关机
    shutdown -a 取消一切定时
    shutdown -s -f -t 秒 定时强制关机
    shutdown -s -f -t 秒 -c “hahaha” 定时强制关机并且备注 hahaha
    shutdown -r -t 秒 定时重启，加上 -f 为强制重启
    shutdown -l 注销，同 logoff 命令
    注意：-s 是关机 -r 重启 -t 时间 -a 所有 -f 强制force -c 备注 -l 注销
    高级命令
    fsutil file createnew d:\system.ini 20480000000 指定位置创建指定大小文件（全是空格的文件）
    attrib +h +s +r 文件夹/文件名.扩展名 增加隐藏（hidden）、系统保护、只读属性（不能修改删除等一系列操作）（+ 改为 - 为取消修改文件属性）
    assoc .txt=exefile 将 txt 文件改成 exe 运行程序
    assoc .txt=txtfile 将 txt 文件改成 txt 文件（上个命令的解药）.txt=xxxfile 固定写法
    ntsd -c q -pn winlogon.exe 蓝屏（只针对于微软 xp/2003 系统）
    taskkill /im explorer.exe /f 强制杀死桌面进程

    批处理（脚本）

    批处理的作用

    自上而下成批的处理每一条命令，直到执行最后一条

    如何创建批处理

    扩展名：.bat
    创建方法：新建一个记事本文件，然后将拓展名改为 .bat

    批处理基本语法

    @echo off 屏蔽执行过程，直接输出结果
    pause 暂停脚本（按任意键继续执行）
    title 标题 设置 cmd 程序左上角的标题
    echo . 空一行
    命令 >nul 2>nul 正确的错误的显示全部屏蔽（1>nul 屏蔽操作成功显示的信息，但是出错还是会显示 2>nul 屏蔽操作失败显示的信息，但是成功还是会显示）
    start 域名/程序路径 打开某个程序或网页
    start 打开 cmd
    %userprofile% userprofile 是一个系统变量，会根据你登录系统的用户来变化相当于C:\Documents and Settings\Administrator（点击开始
    点击所有程序，再右键点击启动，再点击打开复制路径即可），引入变量都需要用百分号
    :区块名字 goto 跳转至的区块 跳转至区块
    以下程序名 1.bat 就是小型病毒，会已知打开 cmd 程序，导致 cpu 占用率百分之百最后死机蓝屏

    set /p 变量名=提示语句 设置变量用户可以输入（用%变量名%引用）
    if "%变量%"="值" goto 区块 if 语句

    以上程序虽然简单，但是黑客可能会嵌套病毒，例如把区块 2，改成这样不知不觉间就中了病毒
