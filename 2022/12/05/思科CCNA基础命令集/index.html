    Cisco IOS

    Cisco Internetwork Operating System（IOS）就是为 Cisco 设备配备的系统软件。它是 Cisco 的一项核心技术，应用于路由器、局域网交换机、小型无线接入点、具有几十个接口的大型路由器以及许多其他设备。

    可以通过多种方法访问 CLI 环境。最常用的方法有：

    • 控制台
    • Telnet 或 ssh
    • 辅助端口

    以太网技术的线缆标准

    • T-568A = 绿白-绿-橙白-蓝-蓝白-橙-棕白-棕
    • T-568B = 橙白-橙-绿白-蓝-蓝白-绿-棕白-棕
    • 标准网线：不同设备用直通线即两头都是 568A 或者都是 568B
    • 交叉网线：同等设备用交叉线即一头为 568A，一头为 568B
    • 全反线（一般用于配置线缆）两端线序刚好相反

    主要模式及基本命令

    注意：Tab无法补全可以加？即可查看命令库

    1. 用户执行模式 “Router>” ，enable 进入特权模式（测试和查询）
    2. 特权执行模式 “Router#” ，configure terminal 进入全局配置模式(在全局配置模式下测试和查询需要在命令前加 do)
    3. 全局配置模式 “Router(config)#” ，interface f0/0(接口号) 进入路由器的接口中，接口号的 f0/0 是 fastEthernet 快速以太网 0号插槽0号接口的意思，s 是 Serial 串行链路口的意思
    4. 接口配置模式 “Router(config-if)#” ，输入命令行 ip address 192.168.1.1 255.255.255.0（子网掩码）即可配置接口 ip地址
    5. no shutdown 激活接口，会显示 changed state to up 即配置成功
    6. exit 返回上一级 end 直接返回特权模式 “Router#”
    7. show running-config 查询所有正在运行的配置
    8. show ip interface brief 查询所有 IP 接口的情况
    9. show ip route 查询路由表
    10. show vlan brief 查看vlan
    11. show interfaces trunk 查看 trunk 接口
    12. hostname r1 设置名字 enable password 123 为进入特权模式设置密码（show running-config可以查看）enable secret 123 同为为进入特权模式设置密码（show running-config不可以查看）
    13. 用 no 命令可以删除你想要删除的配置，例如 no enable password（不要加上自己设置的参数）

    常见报错命令

    % Incomplete command. // 命令不完整
    % Invalid input detected at ‘^’ marker. // 箭头所指字符无法识别
    % Ambiguous command: “s” // 未知的输入

    快捷键

    快捷键 说明
    Tab 填写命令或关键字的剩下部分
    Ctrl+R 重新显示一行
    Ctrl+Z 退出配置模式并返回到执行模式
    向下箭头 用于在前面用过的命令的列表中向前滚动
    向上箭头 用于在前面用过的命令的列表中向后滚动
    Ctrl+Shift+6 用于中断诸如 ping 或 traceroute 之类的 IOS 进程
    Ctrl+C 放弃当前命令并退出配置模式

    关闭会话超时，防止一段时间不操作自动退出

    line console 0
    no exec-timeout
