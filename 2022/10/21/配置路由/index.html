    配置路由

    搭建拓扑图

    配置每台pc的IP地址

    1. 当前pc的ip地址
    2. 子网掩码255.255.255.0
    3. 默认网关接口ip(x.x.x.1)

    配置每个路由接口的ip

    1. en(enable进入特权模式)
    2. conf t(configure terminal进入全局配置模式)
    3. interface 接口
    4. ip address 当前接口ip 255.255.255.0
    5. no shut(no shutdown激活端口)

    配置每个网段的路由协议

    • 静态路由命令格式（默认路由）

      ip route 0.0.0.0 0.0.0.0 连接下一个路由器的的路由接口ip地址/接口

    • RIP 动态路由配置命令

      router rip
      network 网段

    • EIGRP 动态路由配置命令

      router eigrp 1（AS号）
      network 网段

    • OSPF 动态路由配置命令

      router ospf 1
      network 网段 0.0.0.255 area-id

    注意：边界路由需要配置相连网段的路由协议

    配置边界路由的重分布

    （RIP）度量值：距离或跳数（2）
    （OSPF）度量值：cost（100）
    （静态路由） 度量值：无
    （EIGRP）度量值：带宽、延时、可靠性、负载、最大传输单元MTU（1000、50、255、50、1500）

    • 重分布为 rip

      router rip
      redistribute ospf 1 metric 2
      redistribute eigrp 1 metric 2
      redistribute static metric 2
      default-information originate

    • 重分布为 ospf

      router ospf 1
      redistribute rip metric 100 subnets
      redistribute eigrp 1 metric 100 subnets
      redistribute static subnets
      default-information originate

    subnets：重分发非标准网络

    • 重分布为 eigrp

      router eigrp 1
      redistribute rip metric 1000 100 255 1 1500
      redistribute ospf 1 metric 1000 100 255 1 1500
      redistribute static metric 1000 100 255 1 1500
